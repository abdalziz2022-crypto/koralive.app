# دليل نشر تطبيق GoalTime Pro على المتاجر

لقد قمت بتجهيز المشروع باستخدام **Capacitor**. اتبع الخطوات التالية لتحويل الموقع إلى تطبيق حقيقي:

## 1. المتطلبات المحلية
يجب عليك تحميل المشروع على جهازك الشخصي وتثبيت:
- **Node.js**
- **Android Studio** (للأندرويد)
- **Xcode** (للآيفون - يتطلب ماك)

## 2. خطوات تجهيز المشروع
افتح "Terminal" في مجلد المشروع وقم بتشغيل الأوامر التالية:

```bash
# تثبيت المكتبات
npm install

# بناء نسخة الموقع
npm run build

# إضافة منصات التشغيل
npx cap add android
npx cap add ios
```

## 3. فتح المشروع في بيئة التطوير
بعد إضافة المنصات، يمكنك فتحها برمجياً:

```bash
# لفتح أندرويد ستوديو
npx cap open android

# لفتح Xcode
npx cap open ios
```

## 4. أيقونات التطبيق وشاشة البدء
استخدم أداة `capacitor-assets` لتوليد الأيقونات لجميع المقاسات:
1. ضع صورة `icon.png` و `splash.png` في مجلد `assets`.
2. شغّل الأمر:
```bash
npx capacitor-assets generate
```

## 5. النشر
- **Android**: من داخل Android Studio اختر `Build > Generate Signed Bundle / APK`.
- **iOS**: من داخل Xcode اختر `Product > Archive` ثم اتبع خطوات الرفع لـ App Store Connect.

---
**ملاحظة**: الموقع الآن يدعم تقنية **PWA**، مما يعني أن الزائر يمكنه الضغط على "إضافة إلى الشاشة الرئيسية" وسيظهر كأنه تطبيق مثبت تماماً بدون الحاجة للمتاجر حالياً.
